# 專案開發進度

更新日期：2026-04-12

## 1. 目前總結

本專案已完成前後端主要產品骨架，包含病患端、醫師端、管理員端的基本頁面結構，以及後端 API、資料模型、服務層與 WebSocket 架構。

目前狀態可概括為：

- 前端主要頁面與流程已具備，可進行介面展示與大部分流程串接。
- 後端不是空殼，已有完整模組切分與多數 API 路由。
- 部分功能仍處於 `mock` 或 `placeholder` 階段，尚未完全接上真實服務。
- 前端 build 問題已於本次修正，現在可成功產出 `frontend/dist`。

- 多個前端畫面已具備實際 UI，但仍使用 `VITE_ENABLE_MOCK` 控制 mock 模式。
- 認證、報告、病患列表、警示、病患歷史、問診對話等流程，部分已能呼叫 API，部分仍會退回 mock 資料。

## 6. 目前風險

- 專案根目錄本身不是 git repository，暫時無法直接從目前資料夾取得 branch / commit 開發軌跡。
- 前端功能完成度高，但 mock 使用比例仍不低，若直接當作 fully integrated 狀態會高估進度。
- 後端雖然模組完整，但目前驗證主要是靜態結構與語法，尚未完成端到端啟動驗證。
